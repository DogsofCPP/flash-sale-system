## 3. 秒杀核心交互流程

### 3.1 点击抢购按钮

```
用户点击"立即抢购"
        │
        ▼
前端检查：userId 是否存在？
        │ 不存在 → 跳转登录页
        ▼ 存在
发送 POST /api/seckill/seckill
Headers: X-User-Id: {userId}
Body: { activityId, productId, quantity: 1 }
        │
        ▼
显示 loading 遮罩层，按钮文字改为"抢购中..."
        │
        ├─── 成功 (code=200)
        │        │
        │        ▼
        │    显示成功弹窗
        │    订单号: 1720000000001ABC12345
        │    金额: ¥99.00
        │    倒计时: 请在 15 分钟内完成支付
        │        │
        │        ▼
        │    [去支付] [查看订单]
        │
        ├─── 库存不足 (code=1004)
        │        │
        │        ▼
        │    显示"商品已售罄"弹窗
        │    [我知道了]
        │
        ├─── 超限 (code=1005)
        │        │
        │        ▼
        │    显示"您已购买过该商品"弹窗

        │
        └─── 限流 (code=429)
                 │
                 ▼
             显示"请求过于频繁，请稍后再试"
```

### 3.2 倒计时实现

```javascript
function initCountdown(targetTime, elementId) {
    const update = () => {
        const now = Date.now();
        const diff = targetTime - now;

        if (diff <= 0) {
            document.getElementById(elementId).innerText = '活动进行中';
            // 刷新页面获取最新活动状态
            location.reload();
            return;
        }

        const hours = Math.floor(diff / (1000 * 60 * 60));
        const minutes = Math.floor((diff % (1000 * 60 * 60)) / (1000 * 60));
        const seconds = Math.floor((diff % (1000 * 60)) / 1000);

        document.getElementById(elementId).innerText =
            `${String(hours).padStart(2, '0')}:${String(minutes).padStart(2, '0')}:${String(seconds).padStart(2, '0')}`;

        requestAnimationFrame(update);
    };

    update();
}
```

### 3.3 实时库存更新

```javascript
// 每5秒从服务器拉取最新库存
setInterval(async () => {
    const res = await fetch(`/api/seckill/activities/${activityId}/products`);
    const data = await res.json();

    data.data.forEach(product => {
        const el = document.querySelector(`#stock-${product.productId}`);
        if (el) {
            el.innerText = `剩余 ${product.stock} 件`;
            if (product.stock === 0) {
                disableButton(product.productId);
            }
        }
    });
}, 5000);
```

## 5. 响应式断点设计

| 设备类型 | 屏幕宽度 | 商品卡片布局 | 列数 |
|---------|---------|------------|------|
| 超大屏桌面 | > 1400px | 4列 | 4 |
| 标准桌面 | 1024px - 1400px | 3列 | 3 |
| 平板横屏 | 768px - 1024px | 2列 | 2 |
| 手机 | < 768px | 1列 | 1 |

## 6. 前端技术选型

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3 | 页面结构与样式 |
| Vanilla JavaScript | 交互逻辑（无框架依赖） |
| Axios | HTTP 请求 |
| LocalStorage | Token 缓存 |
| CSS Flexbox / Grid | 响应式布局 |
| CSS Animations | 按钮脉冲动画、loading 动画 |
