# 分布式事务

## 核心文件

| 文件 | 说明 |
|------|------|
| `common/.../transaction/TransactionContext.java` | 事务上下文 |
| `common/.../transaction/TccTransaction.java` | TCC事务记录 |
| `common/.../transaction/TransactionMessage.java` | 事务消息 |
| `common/.../transaction/TransactionStatus.java` | 事务状态常量 |
| `common/.../transaction/annotation/*.java` | TCC注解（@TccTry/@TccConfirm/@TccCancel/@TccTransaction） |
| `seckill-service/.../transaction/TccTransactionCoordinator.java` | TCC协调器 |
| `seckill-service/.../transaction/TransactionalSeckillService.java` | TCC秒杀服务 |
| `seckill-service/.../transaction/ReliableMessageService.java` | 可靠消息服务 |
| `order-service/.../service/PaymentService.java` | 支付与订单状态一致性 |

## 架构

- TCC (Try-Confirm-Cancel) + 可靠消息（Kafka）
- 库存预扣（Redis Lua） → 消息异步创建订单 → 最终一致性
- 订单支付（本地事务） → MQ通知 → 状态更新一致性
