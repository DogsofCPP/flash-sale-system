# 所有问题修复总结

## 问题时间线

### 2024-01-15 - Docker镜像拉取失败

**问题：**
confluentinc/cp-kafka:7.5.0 等镜像无法从Docker Hub拉取

**解决：**
配置国内镜像加速器

---

### 2024-01-20 - 编译错误

**问题：**
MyBatis Mapper绑定失败

**解决：**
正确配置 mapper-locations 和 namespace

---

### 2024-01-25 - 数据库连接超时

**问题：**
MySQL容器启动慢，应用连接超时

**解决：**
1. 增加健康检查超时时间
2. 使用 depends_on 的 condition 配置
3. 增加重试机制

---

### 2024-02-01 - Redis连接失败

**问题：**
Redis需要密码但应用未配置

**解决：**
在application.yml中配置：
```yaml
spring:
  redis:
    password: redis123
```

---

### 2024-02-05 - 端口冲突

**问题：**
Docker容器端口与本地服务冲突

**解决：**
修改docker-compose.yml中的端口映射：
```yaml
ports:
  - "3306:3306"  # 改为其他端口如 "33060:3306"
```
