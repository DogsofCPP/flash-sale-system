# 问题修复文档

## 1. 编译问题修复

### 1.1 MyBatis版本兼容性问题

**问题描述：**
```
org.apache.ibatis.binding.BindingException: Invalid bound statement (not found)
```

**原因分析：**
Mapper XML文件中的namespace与Mapper接口不匹配，或XML文件未正确放置。

**解决方案：**
1. 确保Mapper XML文件位于正确目录：
   ```
   src/main/resources/mapper/UserMapper.xml
   ```
2. 确保namespace与接口全限定名一致：
   ```xml
   <mapper namespace="com.example.userservice.mapper.UserMapper">
   ```
3. 在application.yml中配置mapper路径：
   ```yaml
   mybatis:
     mapper-locations: classpath:mapper/*.xml
     type-aliases-package: com.example.userservice.domain
   ```

### 1.2 Spring Boot 3.x 兼容性问题

**问题描述：**
```
java.lang.ClassNotFoundException: javax.persistence.Column
```

**原因分析：**
Spring Boot 3.x 使用 Jakarta EE 9+，不再支持 `javax.*` 命名空间。

**解决方案：**
```xml
<!-- Spring Boot 3.x -->
<dependency>
    <groupId>jakarta.persistence</groupId>
    <artifactId>jakarta.persistence-api</artifactId>
</dependency>
```

### 1.3 Druid连接池配置问题

**问题描述：**
```
druid pool initialize exception
```

**解决方案：**
```yaml
spring:
  datasource:
    druid:
      initial-size: 5
      min-idle: 5
      max-active: 20
      max-wait: 60000
      validation-query: SELECT 1
      test-while-idle: true
      test-on-borrow: false
      test-on-return: false
```
