# 负载均衡配置

## 1. Nginx负载均衡

### 1.1 upstream配置

```nginx
upstream backend {
    server 127.0.0.1:8081;
    server 127.0.0.1:8082;
}
```

### 1.2 负载均衡策略

| 策略 | 配置 | 说明 |
|------|------|------|
| 轮询 | `server 127.0.0.1:8081;` | 默认策略，每个请求依次分配 |
| 权重 | `server 127.0.0.1:8081 weight=2;` | 权重越高，被分配概率越大 |
| IP哈希 | `ip_hash;` | 同一IP固定访问同一后端 |
| 最少连接 | `least_conn;` | 分配给连接数最少的服务器 |

### 1.3 健康检查

```nginx
upstream backend {
    server 127.0.0.1:8081 max_fails=3 fail_timeout=30s;
    server 127.0.0.1:8082 max_fails=3 fail_timeout=30s;
    keepalive 32;
}
```

## 2. 完整Nginx配置

```nginx
user nginx;
worker_processes auto;
error_log /var/log/nginx/error.log warn;
pid /var/run/nginx.pid;

events {
    worker_connections 1024;
}

http {
    include /etc/nginx/mime.types;
    default_type application/octet-stream;

    log_format main '$remote_addr - $remote_user [$time_local] "$request" '
                    '$status $body_bytes_sent "$http_referer" '
                    '"$http_user_agent" "$http_x_forwarded_for"';

    access_log /var/log/nginx/access.log main;
    sendfile on;
    keepalive_timeout 65;

    upstream backend {
        server 127.0.0.1:8081;
        server 127.0.0.1:8082;
        keepalive 32;
    }

    server {
        listen 80;
        server_name localhost;

        location / {
            root /usr/share/nginx/html;
            index index.html;
            try_files $uri $uri/ /index.html;
        }

        location /api/ {
            proxy_pass http://backend/;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            proxy_connect_timeout 60s;
            proxy_send_timeout 60s;
            proxy_read_timeout 60s;
        }

        location /health {
            access_log off;
            return 200 "healthy";
        }
    }
}
```

## 3. 会话保持

### 3.1 IP哈希方式

```nginx
upstream backend {
    ip_hash;
    server 127.0.0.1:8081;
    server 127.0.0.1:8082;
}
```

### 3.2 Cookie方式

```nginx
upstream backend {
    server 127.0.0.1:8081;
    server 127.0.0.1:8082;
}

server {
    # 添加Cookie标记
    add_header Set-Cookie "route=backend";
}
```

## 4. 限流配置

### 4.1 连接数限制

```nginx
limit_conn_zone $binary_remote_addr zone=addr:10m;

server {
    limit_conn addr 10;
}
```

### 4.2 请求速率限制

```nginx
limit_req_zone $binary_remote_addr zone=seckill:10m rate=10r/s;

server {
    location /api/seckill/ {
        limit_req zone=seckill burst=20 nodelay;
    }
}
```

## 5. 测试验证

```bash
nginx -t

# 重载配置
nginx -s reload

# 查看连接状态
curl http://localhost/health
```

## 6. Docker中运行Nginx

```bash
# 启动Nginx
docker run -d \
  --name nginx \
  -p 80:80 \
  -v ./nginx.conf:/etc/nginx/nginx.conf:ro \
  -v ./html:/usr/share/nginx/html:ro \
  nginx:alpine
```
