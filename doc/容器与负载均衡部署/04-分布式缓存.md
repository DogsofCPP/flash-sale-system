# 分布式缓存

## 1. Redis缓存策略

### 1.1 缓存模式

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| Cache Aside | 应用直接操作缓存和数据库 | 通用场景 |
| Read Through | 缓存自动加载数据 | 读多写少 |
| Write Through | 缓存同步写数据库 | 数据一致性要求高 |
| Write Behind | 异步批量写数据库 | 高并发写入 |

### 1.2 Cache Aside模式

```java
public Product getProduct(Long id) {
    // 1. 先查缓存
    Product product = redisTemplate.opsForValue().get("product:" + id);
    if (product != null) {
        return product;
    }

    // 2. 缓存未命中，查数据库
    product = productMapper.findById(id);

    // 3. 写入缓存
    if (product != null) {
        redisTemplate.opsForValue().set("product:" + id, product, 1, TimeUnit.HOURS);
    }

    return product;
}
```

## 2. 缓存问题解决方案

### 2.1 缓存穿透

查询不存在的数据导致频繁查数据库。

**解决方案：布隆过滤器**
```java
private BloomFilter<Long> bloomFilter = BloomFilter.create(
    Funnels.longFunnel(),
    1000000,
    0.01
);

public Product getProduct(Long id) {
    // 布隆过滤器检查
    if (!bloomFilter.mightContain(id)) {
        return null; // 一定不存在
    }
    // 继续查缓存和数据库
}
```

### 2.2 缓存击穿

热点key过期瞬间大量请求击穿到数据库。

**解决方案：互斥锁**
```java
public Product getProduct(Long id) {
    Product product = redisTemplate.opsForValue().get("product:" + id);
    if (product == null) {
        String lockKey = "lock:product:" + id;
        String lockValue = UUID.randomUUID().toString();

        if (redisTemplate.opsForValue()
            .setIfAbsent(lockKey, lockValue, 10, TimeUnit.SECONDS)) {
            try {
                product = productMapper.findById(id);
                if (product != null) {
                    redisTemplate.opsForValue().set("product:" + id, product, 1, TimeUnit.HOURS);
                }
            } finally {
                redisTemplate.delete(lockKey);
            }
        } else {
            // 等待其他线程加载
            Thread.sleep(50);
            return getProduct(id);
        }
    }
    return product;
}
```

### 2.3 缓存雪崩

大量缓存同时过期。

**解决方案：过期时间随机化**
```java
// 基础过期时间1小时，附加随机偏移量
int baseExpire = 3600;
int randomExpire = baseExpire + RandomUtils.nextInt(300);
redisTemplate.opsForValue().set("product:" + id, product, randomExpire, TimeUnit.SECONDS);
```

## 3. Redis数据序列化

```java
@Configuration
public class RedisConfig {
    @Bean
    public RedisTemplate<String, Object> redisTemplate(
            RedisConnectionFactory factory) {
        RedisTemplate<String, Object> template = new RedisTemplate<>();
        template.setConnectionFactory(factory);

        // JSON序列化
        Jackson2JsonRedisSerializer<Object> serializer =
            new Jackson2JsonRedisSerializer<>(Object.class);

        template.setKeySerializer(new StringRedisSerializer());
        template.setValueSerializer(serializer);
        template.setHashKeySerializer(new StringRedisSerializer());
        template.setHashValueSerializer(serializer);

        template.afterPropertiesSet();
        return template;
    }
}
```
