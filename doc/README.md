# 分布式秒杀系统 - 文档中心

> 本项目是一个高性能分布式商品秒杀系统，支持万级并发、防超卖、缓存穿透、读写分离等核心特性。

## 快速开始

### 环境要求

- JDK 17+
- Maven 3.8+
- Docker 20.10+
- MySQL 8.0
- Redis 7.0

### 一键启动

```bash
# 启动所有服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f user-service
```

### 接口测试

```bash
# 测试秒杀接口
curl -X POST http://localhost:8081/api/seckill/order \
  -H "Content-Type: application/json" \
  -d '{"userId":1,"activityId":1,"productId":1}'
```

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 基础框架 | Spring Boot 3.2 | 最新LTS、支持虚拟线程 |
| 微服务 | Spring Cloud 2023 | 服务注册、配置中心 |
| 缓存 | Redis 7.0 + Redisson | 分布式锁、热点缓存 |
| 消息队列 | RabbitMQ 3.12 | 异步解耦、削峰填谷 |
| 数据库 | MySQL 8.0 | 主从复制、读写分离 |
| 搜索引擎 | Elasticsearch 8.12 | 商品全文检索 |
| 容器化 | Docker + K8s | 弹性伸缩、微服务编排 |
